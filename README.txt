Change the default email address and sender name output for all message sent from your WP dashboard.

== Description ==

Custom Email Sender allow you to easily customize the default email sender name and default email address directly from your Wordpress dashboard.

== Installation ==

1. Upload "custom-email-sender" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Click the "Email Sender" link located under your "Settings" section to configure the plugin options.
